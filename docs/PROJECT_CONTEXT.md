## Project Name

The repository is documented as **Vegetable Vision CNN**. The name keeps the original vegetable classification intent while making the project title clearer for future readers.

## Original Work

The original submission notebook is retained as `DELE_CA1_A.ipynb`. It includes the full narrative, experiments, model comparison, training outputs, and evaluation discussion from the CA1 work.

## Repository Modernization

The repository has been expanded with:

- a reusable Python package under `src/vegetable_vision/`
- pytest coverage for deterministic helper logic
- notebook splitting tooling
- structured documentation
- local environment examples
- CI, security scanning, and dependency update automation
- data, model, report, and notebook folders with clear ownership

The modernization does not remove the original notebook. Instead, it makes the project easier to inspect, reproduce, and extend.
