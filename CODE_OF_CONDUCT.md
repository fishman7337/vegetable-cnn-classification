# Code of Conduct

## Our Standards

This project is expected to remain respectful, inclusive, and academically honest. Contributors should:

- use welcoming and professional language
- respect differing experience levels
- give constructive feedback
- credit original work and sources
- avoid harassment, insults, or discriminatory behavior

## Academic Integrity

This repository includes academic coursework. Do not use it to misrepresent authorship, bypass assessment rules, or submit work without proper permission and attribution.

## Enforcement

Unacceptable behavior may result in removal of comments, rejected contributions, or restricted access to the repository.

If you need to report a concern, contact the repository owner privately with relevant details and links.
